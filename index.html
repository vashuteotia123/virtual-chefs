Feels like home baked

Bake it how you want, when you want. Experiment with different flavors virtually and have the tastiest cookie on your doorstep.

Our moto

To inspire virtual baking. Bake it how like it.


The customer is allowed to use ingredients from a preset list of ingredients and also allows the customer to suggest an ingredient if their favourite ingredient is not available. After making the cookie dough, the customer is given the choice to either checkout with the selected cookie dough or to have it baked and delivered.

4 steps

1. Select the ingredients of your choice.

2. Suggest an ingredient if you don't find it.

3. Choose to have it baked or not.

4. Get it when you want it.

About

Virutal Chefs is a web application that allows users to create an account and create a profile. The user can then create a recipe and add it to their profile. The user can then view the recipes that they have created and also view the recipes that are available in the database.
